# FakeHSP-for-Cpp
SDL2を使ったC++向けの関数群 FakeHSP for C++の置き場

# 概要
FakeHSP fot C++は、C++とSDL2ライブラリを用いた、HSP(Hot Soup Processor)風関数群です。  
C++でHSPっぽく、ウィンドウへの画像や文字、図形の描画が可能になります。

# 使い方・導入方法
導入にはSDL2のインストールが必要です。  
使い方と導入方法については、  
http://yotiosoft.com/fakehsp-for-cpp/  
をご覧ください。
